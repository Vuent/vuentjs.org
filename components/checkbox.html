# Checked

# Disabled

# Custom label

<vnt-checkbox>Option</vnt-checkbox>

# Configuration

# checked

Default: false

Decides about the checkbox's state.

# disabled

Default: false

Determines whether the checkbox should be in disabled mode or not.

# label

Default: 'Checkbox'

The checkbox's label text.
